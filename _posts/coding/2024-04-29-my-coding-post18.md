### 문제11
<hr>
다음을 보고 결괏값을 작성하시오!

````javascript
{
        let a, b, result;
        a = 7, b = 4
        result = a | b;
    
        console.log(result)
}
````

이 코드는 JavaScript에서 두 변수 a와 b의 비트 OR 연산(|)을 수행합니다. 여기서 변수 a는 7로, b는 4로 초기화되어 있습니다.

비트 OR 연산은 두 정수의 이진 표현에서 비트 단위로 수행되며, 두 입력 비트 중 적어도 하나가 1이면 결과 비트가 1이 됩니다. 구체적인 계산은 다음과 같습니다.

1) 이진수 변환: <br>
* a = 7 의 이진 표현: 0111<br>
* b = 4 의 이진 표현: 0100
<br>

2) 비트 OR 연산: <br>
* 0111(7) <br>
* 0100(4) <br>
* 위 두 이진수에 대해 비트 OR 연산을 수행하면, 각 비트 위치별로 OR 연산을 적용합니다.

````markdown
0111
| 0100
------
0111
````

3) 이진수 결과를 십진수로 변환: <br>
* 이진수 0111은 십진수로 변환하면 7입니다.
<br>

따라서, console.log(result)에서 <mark>출력되는 값은 7입니다.</mark>

<br>

### 문제12
<hr>
다음을 보고 결괏값을 작성하시오!

````javascript
{
        function solution(a, b, c){
                let answer = "YES", max;
                let total = a + b + c;
        
                if(a > b) max = a;
                else max = b;
                if(c > max) max = c;
                if(total-max <= max) answer = "NO"; 
                
                return answer;
        }
    
        console.log(solution(53, 93, 107));
}
````

이 코드는 세 개의 수 a, b, c를 인수로 받아, 이 수들을 변으로 하는 삼각형이 만들어질 수 있는지를 판단하는 JavaScript 함수 solution입니다. 삼각형의 성립 조건은 어느 한 변의 길이가 다른 두 변의 길이의 합보다 작아야 합니다. 코드의 로직은 다음과 같습니다:

1) 변수 초기화: <br>
* answer는 기본적으로 "YES"로 설정되어 있습니다. 이는 기본적으로 삼각형이 성립한다고 가정합니다.<br>
* max는 세 변 중 가장 긴 변의 길이를 저장하는 데 사용됩니다.<br>
* total은 세 변의 길이 합을 저장합니다.
<br>

2) 가장 긴 변 찾기:
* a, b, c 중에서 가장 큰 값을 max에 할당합니다.
<br>

3) 삼각형 성립 조건 검사:<br>
* total - max는 가장 긴 변을 제외한 두 변의 길이 합입니다.<br>
* 삼각형의 성립 조건은 가장 긴 변의 길이가 다른 두 변의 길이 합보다 작아야 합니다.<br>
* if(total-max <= max) answer = "NO";는 이 조건을 만족하지 않을 경우, 즉 가장 긴 변의 길이가 다른 두 변의 길이 합보다 크거나 같은 경우에 "NO"를 반환합니다.
<br>

4) 특정 입력 값에 대한 결과: <br>
* solution(53, 93, 107)을 호출하면, 가장 긴 변 max는 107입니다. <br>
* 다른 두 변의 합 total - max는 53 + 93 + 107 - 107 = 146입니다. <br>
* 146 (다른 두 변의 합)는 107 (가장 긴 변)보다 크므로 삼각형의 성립 조건을 만족합니다. <br>
* 따라서 answer = "YES"가 유지되고, 이 값이 반환됩니다.

🎈 답
<mark>console.log(solution(53, 93, 107));의 출력은 "YES"입니다. </mark>
이는 제공된 변의 길이로 삼각형을 만들 수 있음을 의미합니다.
<br>

### 문제13
<hr>
다음을 보고 결괏값을 작성하시오!

````javascript
{
        function solution(a, b, c){
                let answer;
                if(a < b) answer = a;
                else answer = b;
                if(c <= answer) answer = c; 
                return answer;
        }
        console.log(solution(15, 12, 11));
}
````
이 JavaScript 함수 solution은 세 개의 매개변수 a, b, c를 받아서 이 중 가장 작은 값을 찾아 반환합니다. 코드의 구체적인 실행 과정은 다음과 같습니다.

1) 가장 작은 값 비교:<br>
* if (a < b) 조건을 평가하여, a가 b보다 작은 경우 answer에 a를 할당하고, 그렇지 않으면 answer에 b를 할당합니다. 이로 인해 answer에는 a와 b 중 더 작은 값이 저장됩니다.<br>

2) 세 번째 값 비교:<br>
* if (c <= answer) 조건을 평가하여, c가 answer에 저장된 값보다 작거나 같은 경우 answer를 c로 업데이트합니다. 이는 c가 현재까지 비교된 두 값보다 작거나 같다면 최소값으로 선택되어야 함을 의미합니다.
<br>

3) 결과 반환:<br>
* 함수는 최종적으로 answer를 반환합니다. 이 값은 입력된 세 값 중 가장 작은 값입니다.
<br>

4) 특정 입력 값에 대한 결과 (solution(15, 12, 11)):<br>
* 처음 비교에서 a는 15, b는 12입니다. 12 < 15이므로 answer는 12가 됩니다. <br>
* 두 번째 비교에서 c는 11로, answer의 현재 값인 12보다 작습니다. 그러므로 answer는 11로 업데이트됩니다. <br>

🎈 답<br>
<mark>console.log(solution(15, 12, 11));의 출력은 11입니다. </mark>
이는 세 수 15, 12, 11 중 가장 작은 수인 11이 최종 결과로 반환됩니다.
<br>

### 문제14
<hr>
다음을 보고 결괏값을 작성하시오!

````javascript
{
        function solution(day, arr){
                let answer = 0;
                for(let x of arr){
                        if(x % 10 == day) answer++;
                }
                return answer;
        }
        
        arr = [25, 23, 11, 47, 53, 17, 33, 40];
        console.log(solution(0, arr));
}
````

이 JavaScript 코드 조각은 solution 함수를 정의하고, 그 함수를 호출하여 결과를 콘솔에 출력합니다.<br>
함수 solution은 두 개의 인자를 받습니다: 하나는 day이고, 다른 하나는 arr 배열입니다.<br>
함수의 목적은 배열 arr에서 각 요소의 일의 자리 숫자가 day와 동일한 요소의 수를 세는 것입니다.<br>
<br>
여기서 arr = [25, 23, 11, 47, 53, 17, 33, 40] 배열이 주어졌고, day = 0으로 설정되었습니다.<br>
따라서 배열 내의 각 요소를 10으로 나눈 나머지가 0인 요소의 수를 세야 합니다.<br>
배열에서 일의 자리 숫자가 0인 요소는 40 한 개뿐입니다.<br>
<br>
결과적으로, <mark>console.log(solution(0, arr));는 1을 출력합니다.</mark>
는 배열 arr 내에서 일의 자리 숫자가 0인 요소가 하나 있기 때문입니다.

````javascript
{
        let a, b, result;
        a = 7, b = 4
        result = a & b;
    
        console.log(result);
}
````

이 코드에서는 두 변수 a와 b에 각각 7과 4의 값을 할당하고, 이 두 수를 비트 AND 연산(&)을 수행합니다. 비트 AND 연산은 두 입력의 비트가 모두 1일 경우 결과 비트가 1이 됩니다.<br>
<br>
7과 4의 이진 표현은 다음과 같습니다:<br>
<br>
7: 0111<br>
4: 0100
<br>
각 비트를 AND 연산하면:<br>
<br>
0111<br>
0100<br>
<hr>
0100
<br>
0100은 이진수로 4입니다. 그러므로 결과값 result는 4이며, <mark>console.log(result);는 4를 콘솔에 출력합니다.</mark>

### 문제16
<hr>
다음을 보고 결괏값을 작성하시오!

````javascript
{
        let a = 6, b = 9, c = 3, result;
        result = ++a + b++ + ++c;
    
        console.log(result);
        console.log(a+b+c);
}
````

이 코드에서는 여러 변수에 초기값을 할당하고, 이들에 대해 증가 연산자(++)를 사용한 후, 덧셈을 수행합니다.
연산자 ++는 변수의 값을 1 씩 증가시키며, 변수 앞에 위치하면 값을 증가시킨 후에 평가하고, 변수 뒤에 위치하면 값을 평가한 후에 증가시킵니다.
<br>
초기 값:<br>
<br>
a = 6<br>
b = 9<br>
c = 3<br>
<br>
✏ result 계산:<br>
<br>

* ++a는 a를 먼저 1 증가시키므로 a는 7이 됩니다.<br>
* b++는 b를 평가한 후에 증가시키므로, 이 연산의 값은 9이고, 이후 b는 10이 됩니다.<br>
* ++c는 c를 먼저 1 증가시키므로 c는 4가 됩니다.<br>
따라서, result = 7 + 9 + 4 = 20.<br>

<mark>console.log(result);의 출력은 20입니다.</mark>
<br>
두 번째 console.log(a + b + c);에서는 모든 변수의 최종 값:<br>
<br>

* a = 7<br>
* b = 10<br>
* c = 4<br>
* a + b + c = 7 + 10 + 4 = 21.

따라서, <mark>두 번째 console.log의 출력은 21입니다.</mark>

💡 여기서는, ++이 변수앞에 붙으면 1을 먼저 더해주고, 변수 뒤에 ++이 붙으면 초기값은 변수 그대로의 값이 출력되고,<br>
나중에 값을 더해준다는 것이 중요 풀이인것같다.
++이 변수 뒤에 붙을 때는 그 변수를 먼저 사용하고, 그 다음에 해당 변수의 값을 1 증가시킨다.
<br>

### 문제17
<hr>
다음을 보고 결괏값을 작성하시오!

````javascript
{
        let a, b = 10;

        for(let i=0; i<5; i++){
            a = i;
            b -= a;
        }
        console.log(a, b)
}
````
이 코드 블록은 a와 b 두 변수를 사용하고 있으며, b는 초기값으로 10이 할당되어 있습니다.<br>
그리고 for 루프를 통해 5번 반복을 실행하면서 다음과 같은 연산을 수행합니다:<br>
<br>
변수 a는 루프의 인덱스 i로 설정되고,<br>
변수 b는 a의 현재 값만큼 감소합니다.<br>
루프는 i가 0부터 시작해서 4까지 실행됩니다. 각 반복에서의 a의 값과 b에서 a 값을 뺀 후의 결과는 다음과 같습니다:
<br>

1) 첫 번째 반복 (i=0): a = 0, b = 10 - 0 = 10<br>
2) 두 번째 반복 (i=1): a = 1, b = 10 - 1 = 9<br>
3) 세 번째 반복 (i=2): a = 2, b = 9 - 2 = 7<br>
4) 네 번째 반복 (i=3): a = 3, b = 7 - 3 = 4<br>
5) 다섯 번째 반복 (i=4): a = 4, b = 4 - 4 = 0<br>

루프 종료 후, a는 4이고 b는 0입니다. 따라서 출력 결과는:
<mark>4, 0</mark>
<br>

### 문제18
<hr>
다음을 보고 결괏값을 작성하시오!

````javascript
{
        let num = 10;
        num += 2;
        num -= 3;
        num *= 5;
        num /= 5;
        num %= 9;
        
        console.log(num)
}
````

이 코드에서 변수 num은 여러 산술 연산자를 사용하여 값이 계속 변경됩니다. 각 단계의 연산을 차례대로 살펴보면 다음과 같습니다:<br>

1) 초기값 설정: num = 10<br>
2) 더하기 연산: num += 2 -> num = 10 + 2 = 12<br>
3) 빼기 연산: num -= 3 -> num = 12 - 3 = 9<br>
4) 곱하기 연산: num *= 5 -> num = 9 * 5 = 45<br>
5) 나누기 연산: num /= 5 -> num = 45 / 5 = 9<br>
6) 나머지 연산(=모듈로 연산): num %= 9 -> num = 9 % 9 = 0<br>
마지막 연산 후 num의 값은 0입니다. 따라서 <mark>출력 결과는 0입니다.</mark>
<br>

````javascript
{
        let num = [1, 5, 1, 2, 7, 5];
        for(let i=0; i<6; i++){

            }
````

배열은 [1, 5, 1, 2, 7, 5]이며, for 루프는 0부터 5까지의 인덱스를 반복합니다.<br>
조건 if((i+2) % 2 == 0)는 인덱스 i에 2를 더한 후 그 나머지가 짝수인지 확인합니다.
<br>

1) i = 0: 2 % 2 == 0 -> true -> 출력: 1<br>
2) i = 1: 3 % 2 == 1 -> false<br>
3) i = 2: 4 % 2 == 0 -> true -> 출력: 1<br>
4) i = 3: 5 % 2 == 1 -> false<br>
5) i = 4: 6 % 2 == 0 -> true -> 출력: 7<br>
6) i = 5: 7 % 2 == 1 -> false<br>
따라서 짝수 인덱스(0, 2, 4)에 해당하는 배열 요소가 출력되며, 최종 결과는:<br>
<mark>1</mark>
<mark>1</mark>
<mark>7</mark>입니다.
<br>

### 문제20
<hr>
다음을 보고 결괏값을 작성하시오!

````javascript
{
        let data = [10, 6, 7, 9, 3];
        let temp;
    
        for(let i=0; i<4; i++){
            for(let j=i+1; j<5; j++){
                if(data[i] > data[j]){
                    temp = data[i];
                    data[i] = data[j];
                    data[j] = temp;
                }
            }
        }
    
        console.log(data) 
}
````
이 코드는 주어진 배열 data를 오름차순으로 정렬하는 버블 정렬(bubble sort) 알고리즘의 변형을 수행합니다.<br>
배열 내의 요소들을 순회하면서, 현재 요소가 다음 요소보다 클 경우 두 요소의 위치를 바꿉니다.<br>
이 과정을 배열의 모든 요소에 대해 반복하여 전체 배열을 정렬합니다.<br>
<br>
정렬 과정은 다음과 같습니다:
<br>

1) 첫 번째 요소인 10과 그 뒤의 요소들(6, 7, 9, 3)을 비교하고 필요에 따라 위치를 교환합니다. 이 과정에서 10은 3과 위치가 바뀌어 배열은 [3, 6, 7, 9, 10]이 됩니다.
2) 이제 두 번째 요소인 6은 이미 올바른 위치에 있어 변경할 필요가 없습니다.
3) 세 번째 요소인 7도 올바른 위치에 있습니다.
4) 네 번째 요소인 9도 이미 올바른 위치에 있습니다.
5) 따라서 정렬 후의 최종 배열은 다음과 같습니다:
<br>
<mark>[3, 6, 7, 9, 10]</mark>
